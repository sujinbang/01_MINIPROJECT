# 부산 관광산업 활성화 방안
<br>

## ・프로젝트 개요

본 프로젝트는 부산 관광객 유치를 위해 부산의 여행 상품(트래킹 + 세븐브릿지)을 홍보하고 전통시장 활성화 및 관광객을 위한 여행 정보 시각화를 목적으로 진행된 프로젝트 입니다.
EDA 기반의 데이터 분석 및 시각화를 진행하였으며. 이 때 사용된 기술은 pandas, matplotlib, seaborn, folium입니다.

## ・데이터 출처

1. 관광객 수 현황 분석 : 공공데이터포털, 통계철
2. 전통시장, 음식점 카페 상권 분석 : 공공데이터포털, 소상공인시장진흥공단, 통계청
